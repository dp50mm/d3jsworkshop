Excercise 1: Shapes

Step 1: create an svg element

The first step for creating visualisations with d3.js is often to create an svg element in your web page. That will the first step of this excercise. In the script add the following piece of code: 


var svg = d3.select('#exercise-1-svg-container')
    .append('svg')
    .attr('width',500)
    .attr('height',500);

Step 2: Add a shape

Now that you have the SVG element as a variable in javascript you can add shapes to it. Try adding a rectangle, circle or other shapes. Play around with it, combining multiple shapes to make some sort of picture. 


d3.append('rect')
    .attr('width',20)
    .attr('height',20)
    .attr('x',20)
    .attr('y',20);

Step 3: Add some fancy styling

Attributes are not just for positioning, through attributes you can change also change fill color and borders. 


svg.append('rect')
   .attr('width',20)
   .attr('height',20)
   .attr('x',100)
   .attr('y',20)
   .attr('fill','green');

Step 4: use a loop

Now that you've gotten some practice with adding individual shapes, it is time to make it more interesting. Using a loop you can easily repeat the same piece of code multiple times. 


for(var i = 0; i < 10; i++) {
    svg.append('rect').attr('width',20)
        .attr('height',i*4)
        .attr('x',i*25)
        .attr('y',100);
}

Step 5: use multiple loops!

It is time to make it even more interesting. Use multiple loops and be creative! 


for(var i = 0; i < 10; i++) {
    for(var j = 0; j < 10; j++) {
        svg.append('rect').attr('width',j*2)
            .attr('height',i*4)
            .attr('x',i*2)
            .attr('y',100-j*10);
    }
}
